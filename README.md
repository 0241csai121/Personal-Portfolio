# PRODIGY_WD_04 - Personal Portfolio Website

## 📌 Description
This project was developed as part of the Prodigy InfoTech Web Development Internship.

The Personal Portfolio Website is built using HTML, CSS, and JavaScript. It is designed to showcase personal information, technical skills, projects, achievements, resume, and contact details through a modern, responsive, and user-friendly interface.

## ✨ Features
- Responsive Design
- Home Section
- About Me Section
- Skills Section
- Projects Showcase
- Resume Download Option
- Contact Information
- Smooth Navigation
- Modern User Interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
PRODIGY_WD_04/
│── index.html
│── style.css
│── script.js
│── images/
│── resume.pdf (optional)


## 🚀 How to Run
1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html` in your browser or use the Live Server extension.
